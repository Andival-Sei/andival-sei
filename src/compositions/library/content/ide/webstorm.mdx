# WebStorm

WebStorm — это мощная интегрированная среда разработки (IDE) от компании JetBrains, специально созданная для веб-разработки. Она предоставляет разработчикам все необходимые инструменты для эффективной работы с JavaScript, TypeScript, HTML, CSS и современными веб-фреймворками. WebStorm известна своим интеллектуальным анализом кода, мощными инструментами рефакторинга и глубокой интеграцией с экосистемой веб-разработки.

## Основные возможности

### Интеллектуальное автодополнение и анализ кода

WebStorm анализирует ваш код в реальном времени и предлагает контекстно-зависимые подсказки, что ускоряет процесс написания кода и снижает вероятность ошибок.

**Особенности:**
- Автодополнение на основе контекста всего проекта
- Анализ типов TypeScript и JavaScript
- Обнаружение потенциальных ошибок до выполнения кода
- Предложения по оптимизации и улучшению кода

### Встроенная поддержка фреймворков

WebStorm предоставляет встроенную поддержку для популярных веб-фреймворков и библиотек:

- **React** — автодополнение JSX, поддержка хуков, анализ пропсов
- **Vue.js** — поддержка Single File Components, автодополнение директив
- **Angular** — полная поддержка TypeScript, шаблонов и декораторов
- **Next.js** — автоматическое создание конфигураций запуска (2025.1+)
- **Nuxt** — интеграция с Nuxt CLI для Vue-проектов
- **Node.js** — отладка, профилирование, поддержка npm/yarn/pnpm

### Интегрированный отладчик

Встроенный отладчик JavaScript позволяет тестировать и исправлять ошибки непосредственно в IDE.

**Возможности:**
- Установка точек останова в коде
- Пошаговое выполнение (Step Over, Step Into, Step Out)
- Просмотр значений переменных и выражений
- Отладка Node.js приложений
- Отладка в браузере через Chrome DevTools Protocol

### Инструменты рефакторинга

WebStorm предлагает широкий спектр инструментов для безопасного рефакторинга кода:

- Переименование переменных, функций, классов с автоматическим обновлением всех ссылок
- Извлечение методов, переменных, констант
- Встраивание переменных и методов
- Перемещение файлов и классов с автоматическим обновлением импортов
- Безопасное удаление неиспользуемого кода

## Live Templates (Шаблоны кода)

WebStorm включает множество встроенных шаблонов и позволяет создавать собственные.

### Популярные шаблоны

**JavaScript/TypeScript:**
- `log` → `console.log()`
- `clg` → `console.log()`
- `for` → цикл for
- `fori` → цикл for с индексом
- `fore` → цикл forEach
- `rfc` → React функциональный компонент
- `usf` → React функциональный компонент с хуками

**HTML:**
- `div` → `<div></div>`
- `a` → `<a href=""></a>`
- Использование Emmet (встроен)

### Создание собственного шаблона

```text
Settings → Editor → Live Templates
  → Нажать "+" → Live Template
  → Указать:
    - Abbreviation (сокращение)
    - Description (описание)
    - Template text (текст шаблона)
    - Context (где применять)
```

**Пример шаблона для React компонента:**

```typescript
import React from 'react';

interface $COMPONENT_NAME$Props {
  $END$
}

export const $COMPONENT_NAME$: React.FC<$COMPONENT_NAME$Props> = () => {
  return (
    <div>
      $END$
    </div>
  );
};
```

Где `$COMPONENT_NAME$` и `$END$` — переменные шаблона.

## Полезные советы

### 1. Используйте Search Everywhere

`Double Shift` открывает универсальный поиск, который ищет:
- Файлы
- Классы и функции
- Настройки
- Действия (Actions)

### 2. Настройте Code Style для команды

```text
Settings → Editor → Code Style
  → Export (экспорт настроек)
  → Создать файл .editorconfig в корне проекта
```

Или используйте `.editorconfig` для синхронизации стиля кода.

### 3. Используйте Local History

WebStorm автоматически сохраняет историю изменений файлов:

```text
File → Local History → Show History
  - Просмотр всех изменений файла
  - Восстановление предыдущих версий
  - Сравнение версий
```

### 4. Настройте File Watchers

Автоматический запуск задач при изменении файлов:

```text
Settings → Tools → File Watchers
  → "+" → Выбрать тип (SCSS, TypeScript, etc.)
  → Настроить команду и аргументы
```

### 5. Используйте Scratch Files

Быстрое создание временных файлов для экспериментов:

```text
Ctrl+Alt+Shift+Insert  - Создать scratch file
  - Выбрать язык
  - Экспериментировать без сохранения
  - Запускать код прямо из scratch файла
```

### 6. Настройте TODO и закладки

```text
Ctrl+Shift+A → "TODO" - Показать все TODO комментарии
F11                    - Добавить закладку
Ctrl+F11               - Закладка с мнемоникой
Shift+F11              - Показать все закладки
```
